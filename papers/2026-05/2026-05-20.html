PEEK: Context Map as an Orientation Cache for Long-Context LLM Agents ★ 高度相关
Zhuohan Gu, Qizheng Zhang, Omar Khattab, Samuel Madden · 2026-05-20
cs.AIagent efficiencycontext cachellm agent
PEEK caches an LLM agent's understanding of recurring external context as a structured "orientation map" embedded in the prompt, maintained by a Distiller, Cartographer, and Evictor that holds a fixed token budget. Across reasoning, aggregation, and context-learning tasks it beats strong baselines by 6.3-34% while using 93-145 fewer agent iterations and incurring 1.7-5.8x lower cost than the SOTA ACE framework.
中文摘要
PEEK 为长上下文 LLM 智能体引入“方向感缓存”：用 Distiller、Cartographer、Evictor 三个组件提炼可迁移知识、生成结构化更新、并维持固定 token 预算，把上下文理解以结构化地图形式嵌入提示。在推理与聚合任务上比强基线提升 6.3-34%，迭代轮次减少 93-145，且较 SOTA 提示学习方法 ACE 成本降低 1.7-5.8 倍，对智能体通信与上下文开销优化具有直接价值。

AutoResearchClaw: Self-Reinforcing Autonomous Research with Human-AI Collaboration ★ 高度相关
Jiaqi Liu, Shi Qiu, Mairui Li, Bingzhou Li et al. · 2026-05-20
cs.AImulti-agent debateagent collaborationhuman-in-the-loop
AutoResearchClaw frames autonomous research as iterative by combining multi-agent debate for hypothesis generation, a self-healing executor that converts failures into information, verifiable result reporting, seven human-in-the-loop intervention modes, and cross-run evolution that turns past mistakes into safeguards. It reports a 54.7% gain over a comparable system and finds targeted collaboration at key decision points outperforms both full autonomy and exhaustive oversight.
中文摘要
AutoResearchClaw 将自主科研建模为迭代过程，集成五大机制：多智能体辩论生成假设、可自愈执行器把失败转化为信息、可验证结果汇报杜绝数据造假、提供七种人机协同干预模式、以及跨实验进化把以往错误固化为安全护栏。系统较同类方法提升 54.7%，并发现“关键决策点定向协作”优于完全自动化或全程审查，对多智能体协同效率研究有借鉴价值。

Harnessing LLM Agents with Skill Programs ◆ 相关
Hongjun Liu, Yifei Ming, Shafiq Joty, Chen Zhao · 2026-05-20
cs.AIllm agentskill library
HASP upgrades reusable skills for LLM agents from passive textual advice into executable Program Functions (PFs) that act as guardrails — activating on failure-prone states to modify the next action or inject corrective context. Applied at inference time, post-training, or in self-improving evolution, HASP improves ReAct-style web search by 25% and Search-R1 by 30.4% on agentic tasks.
中文摘要
本文提出 HASP 框架，将 LLM 智能体从历史经验中提炼的技能升级为可执行的程序函数（PFs），充当主动护栏：在易失败状态触发并修改下一步动作或注入纠错上下文，弥补了传统纯文本经验只能被动建议的不足。HASP 可用于推理时干预、后训练监督或自我进化，在网页搜索任务上较 ReAct 多轮基线提升 25%，相比 Search-R1 在后训练+受控演化下提升 30.4%。

OpenComputer: Verifiable Software Worlds for Computer-Use Agents ◆ 相关
Jinbiao Wei, Qianran Ma, Yilun Zhao, Xiao Zhou et al. · 2026-05-20
cs.AIcomputer-use agent
OpenComputer builds a verifier-grounded environment for computer-use agents, combining hard-coded application state verifiers, a self-improving verification layer, automated task generation, and an evaluation harness across 33 desktop apps and 1,000 tasks. The authors show hard-coded verifiers align with human adjudication better than LLM-as-judge, and that even strong agents (and especially open-source ones) struggle with end-to-end task completion.
中文摘要
OpenComputer 为计算机使用类智能体搭建“可验证的软件世界”：通过应用级硬编码状态验证器、自改进验证层、任务自动生成流水线及评测框架，覆盖 33 款桌面应用与 1000 项任务。实验表明硬编码验证器比 LLM-as-judge 更贴近人工裁决；当前先进智能体在端到端完成上仍困难重重，开源模型差距尤其明显，揭示了真实计算机自动化的稳健性挑战。

EnvFactory: Scaling Tool-Use Agents via Executable Environments Synthesis and Robust RL ◆ 相关
Minrui Xu, Zilin Wang, Mengyi Deng, Zhiwei Li et al. · 2026-05-20
cs.CLtool-use agentagentic rl
EnvFactory automates both halves of agentic-RL data: it discovers and verifies stateful executable tool environments from authentic resources, and synthesizes natural multi-turn trajectories with implicit user intents via topology-aware sampling. Using only 85 environments across 7 domains and 2,575 trajectories, it lifts Qwen3 by up to +15% on BFCLv3 and +8.6% on MCP-Atlas — outperforming prior work that uses ~5x more environments.
中文摘要
EnvFactory 针对智能体强化学习的两大瓶颈（缺乏可扩展的可执行环境、缺乏蕴含隐式人类意图的真实训练数据）提出全自动方案：从真实资源中自主探索并验证有状态的工具环境，并通过拓扑感知采样与校准式精炼合成自然的多轮交互轨迹。仅用 7 个领域 85 个环境合成 2575 条轨迹，便将 Qwen3 在 BFCLv3 提升最多 15%、MCP-Atlas 提升 8.6%，效率超过使用 5 倍环境的同类方法。

Aurora: Unified Video Editing with a Tool-Using Agent ◆ 相关
Yongsheng Yu, Ziyun Zeng, Zhiyuan Xiao, Zhenghong Zhou et al. · 2026-05-20
cs.CVtool-using agent
Aurora pairs a tool-augmented VLM agent with a unified video-diffusion transformer: the agent converts ambiguous user requests into structured edit plans and reference selections that match what the diffusion model expects. Trained with SFT for planning plus preference pairs for tool use, it outperforms text-only baselines and transfers to other frozen video editors; the authors release AgentEdit-Bench for underspecified editing.

Aurora 在统一视频扩散 Transformer 之前接入一个工具增强型 VLM 智能体：该智能体将含混的用户请求转化为结构化编辑计划与参考选择，提前消解文本与视觉歧义。通过 SFT 训练编辑规划与参考选择、用偏好数据训练工具调用与指令润色，Aurora 优于纯文本基线，并能迁移到其他冻结的视频编辑模型；作者还发布了用于评估欠规约编辑的 AgentEdit-Bench。

OmniGUI is a smartphone GUI-agent benchmark of 709 expert episodes / 2,579 action steps across 29 apps that adds synchronized audio and video clips to the usual static screenshots. CopT: Contrastive On-Policy Thinking with Continuous Spaces for General and Agentic Reasoning ◆ 相关
Dachuan Shi, Hanlin Zhu, Xiangchi Yuan, Wanjia Zhao et al. · 2026-05-20
cs.CLagentic reasoningtoken efficiency
CopT reverses the standard CoT order: it produces a draft answer first, then runs on-policy thinking conditioned on it, gating extra deliberation via a contrastive KL estimator that compares the model's support for tokens under discrete vs continuous-embedding inputs. Without extra training, it lifts peak accuracy by up to 23% and cuts token usage by up to 57% on math, coding, and agentic reasoning benchmarks.
中文摘要
CopT 颠覆传统“先思考后回答”的链式推理范式：先产生草稿答案，再以草稿为条件进行同策略反思；并通过对比离散 token 与连续嵌入输入下模型支持度的 KL 估计器作为可靠性验证器，决定是否需要进一步思考。无需额外训练即可在数学、代码及智能体推理任务上将峰值准确率提升 23%、token 用量降低 57%，对降低推理代币开销具有重要意义。
