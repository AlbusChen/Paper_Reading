KVServe: Service-Aware KV Cache Compression for Communication-Efficient Disaggregated LLM Serving ★ 高度相关
Zedong Liu, Xinyang Ma, Dejun Luo, Hairui Zhao et al. · 2026-05-23
cs.DCcs.AIcs.NI
KVServe is a service-aware, adaptive KV cache compression framework for disaggregated LLM serving, where KV state must cross network and storage boundaries and becomes the end-to-end bottleneck. It unifies KV compression into a modular strategy space, uses a Bayesian Profiling Engine to distill a 3D Pareto candidate set (50× less offline search), and combines an analytical latency model with a bandit for online selection — yielding up to 9.13× JCT speedup in PD-separated serving and 32.8× TTFT reduction in KV-disaggregated serving.
中文摘要
在解耦式 LLM 推理服务（PD 分离 / KV 状态分离）中，KV 状态成为跨网络与存储的显式负载，是端到端瓶颈，而静态压缩配置在变化的负载与带宽下并不稳健。KVServe 首次提出服务感知、可自适应的 KV 通信压缩框架：统一压缩策略空间、用贝叶斯剖析引擎搜索并蒸馏 3D Pareto 候选（离线开销降低 50×），并结合解析延迟模型与轻量 bandit 做在线选择，在解耦服务下取得最高 9.13× JCT 加速与 32.8× TTFT 降低。对「多智能体/多服务通信效率」是直接相关工作。

Efficient Agentic Reasoning Through Self-Regulated Simulative Planning ★ 高度相关
Mingkai Deng, Jinyu Hou, Lara Sá Neves, Varad Pimpalkhute et al. · 2026-05-23
cs.AIcs.CLcs.LG
Decomposes agent decision-making into three systems: simulative reasoning (planning via an LLM world model), self-regulation (learned configurator deciding when/how deeply to plan), and reactive execution. The resulting SR²AM (v1.0-30B) matches 685B–1T-parameter agentic LLMs on math, science, tabular, and web search tasks while using 25.8–95.3% fewer reasoning tokens; RL teaches it to plan further ahead rather than more often.
中文摘要
本文针对当前智能体「无差别地长链推理」造成的 token 浪费问题，提出将决策分解为三层系统：基于 LLM 世界模型的模拟推理（System II）、决定何时与如何规划的自我调控（System III）以及反应式执行（System I）。所实现的 SR²AM（v1.0-30B）在数学、科学、表格分析和网页检索任务上 Pass@1 可比肩 685B–1T 量级的智能体大模型，但推理 token 减少 25.8%–95.3%；强化学习显著提升了规划「深度」而非「频次」，对多智能体效率研究具有重要启发。

Maestro: Reinforcement Learning to Orchestrate Hierarchical Model-Skill Ensembles ★ 高度相关
Jinyang Wu, Guocheng Zhai, Ruihan Jin, Yuhao Shen et al. · 2026-05-23
cs.LGcs.CL
Maestro reframes multimodal agent tasks as sequential orchestration over a hierarchical registry of frozen expert models and skills, training a lightweight (4B) policy with outcome-based RL — no step-level supervision. Achieves 70.1% average across ten multimodal benchmarks, surpassing GPT-5 (69.3%) and Gemini-2.5-Pro (68.7%), and the learned policy generalises to unseen out-of-domain experts without retraining.
中文摘要
Maestro 将异质多模态任务建模为对「冻结的专家模型 + 两层技能库」的顺序编排问题，用基于最终结果的强化学习训练一个仅 4B 的轻量策略，无需步骤级监督，便能动态决定何时调用哪一专家–技能组合并适时终止。在 10 个多模态基准上取得 70.1% 的平均准确率，超越 GPT-5 与 Gemini-2.5-Pro；且学习到的协作策略能在不重训的情况下泛化到新加入的领域外专家，对多智能体协同与编排研究具有直接借鉴价值。

Qisheng Su, Zhen Fang, Shiting Huang, Yu Zeng et al. · 2026-05-23
cs.CLcs.AI
ACC (Agent Context Compilation) turns multi-turn agent trajectories (search, software engineering, database) into long-context QA pairs that combine the original question with tool responses and environment observations, training the model to answer directly without tool use. Yields scalable long-context SFT data without annotation; trains Qwen3-30B-A3B to 68.3 on MRCR (+18.1) and 77.5 on GraphWalks (+7.6), comparable to Qwen3-235B-A22B.

Haoran Zhang, Luxin Xu, Zhilin Wang, Runquan Gui et al. · 2026-05-23
cs.AI
π-Bench evaluates proactive personal-assistant agents in long-horizon, multi-turn workflows: 100 tasks across 5 personas with hidden user intents, inter-task dependencies, and cross-session continuity. 中文摘要
