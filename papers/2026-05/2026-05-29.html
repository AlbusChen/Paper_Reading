When Cloud Agents Meet Device Agents: Lessons from Hybrid Multi-Agent Systems ★ 高度相关
· 2026-05-29
multi-agent
Studies hybrid multi-agent systems that combine cloud-hosted frontier LLMs with on-device small language models, where task accuracy, monetary cost, and edge energy consumption are tightly coupled. The authors adapt two representative MAS architectures for hybrid inference and map the Pareto frontier across power, cost, and performance. Their key finding is that the optimal architecture is highly task-dependent, and more frontier-level compute does not consistently translate to better performance.
中文摘要
本文系统研究了将云端前沿大模型与端侧小模型结合的混合多智能体系统，指出任务精度、金钱成本与边缘能耗三者紧密耦合。作者改造了两种代表性多智能体架构以支持混合推理，并刻画了性能-成本-能耗的帕累托前沿。研究发现最优架构高度依赖具体任务，且更强的云端算力并不总能带来更好的性能。

Discovering Cooperative Pipelines: Autoresearch for Sequential Social Dilemmas ★ 高度相关
· 2026-05-29
Proposes a two-level autoresearch framework in which an outer-loop AI agent autonomously redesigns the inner-loop pipeline of an LLM policy-synthesis system for multi-agent Sequential Social Dilemmas. The discovered pipelines reliably exceed hand-designed baselines, sharply tighten run-to-run variance, and outperform prompt-only optimization. Notably, fairness mechanisms emerge only when optimizing for maximin (fairness) objectives rather than efficiency, revealing objective-dependent coordination behavior.
中文摘要
本文提出了一种两层"自动研究"框架：外层AI智能体自主重新设计内层LLM策略合成系统的流程，以应对多智能体序贯社会困境。所发现的流程稳定超越人工设计的基线，显著降低多次运行间的方差，并优于仅靠提示优化的方法。值得注意的是，公平性机制仅在以maximin(公平)目标优化时自发涌现，而在效率目标下不会出现，揭示了协调行为对目标的依赖性。

Towards Verifiable Multimodal Deep Research: A Multi-Agent Harness for Interleaved Report Generation ◆ 相关
· 2026-05-29
multi-agent
Introduces PTAH, a multi-agent harness that orchestrates planning, research, and writing agents to produce verifiable interleaved text-and-image deep-research reports. A Verifier agent enforces factual grounding, citation fidelity, and cross-modal consistency, while a visual working memory keeps images source-aligned to specific claims. Under the new PTAHEval protocol, PTAH produces more reliable and visually informative reports than strong baselines.
中文摘要
本文提出PTAH——一个编排规划、检索与写作智能体的多智能体框架，用于生成可验证的图文交错深度研究报告。其中验证者智能体强制保证事实依据、引用一致性与跨模态一致性，视觉工作记忆则使图像与具体论点的来源对齐。在新提出的PTAHEval评测协议下，PTAH生成的报告比强基线更可靠、信息量更丰富。

AsyncTool: Evaluating the Asynchronous Function Calling Capability under Multi-Task Scenarios ◆ 相关
· 2026-05-29
Presents AsyncTool, a benchmark for evaluating LLM agents' asynchronous function-calling in multi-task settings with realistic tool-response latency, a temporal dimension overlooked by prior evaluations. It provides step-, sub-task-, and task-level efficiency metrics for task switching, dependency tracking, and state maintenance. Results show delayed tool feedback causes clear performance degradation, and agents strong at coordination perform best.
中文摘要

UI-KOBE: Knowledge-Oriented Behavior Exploration for Lightweight Graph-Guided GUI Agents ◆ 相关
· 2026-05-29
UI-KOBE enables lightweight on-device GUI agents by replacing reliance on large VLMs with app-specific graph knowledge: it autonomously explores apps to build knowledge graphs whose nodes are UI states and edges are transitions. At runtime a small agent locates the current node and chooses among self-loops, neighbor transitions, task completion, or fallback actions, lowering inference cost and protecting on-device data. This graph-guided design reduces the planning burden on lightweight models while improving task effectiveness and interpretability.
中文摘要
UI-KOBE通过应用特定的图知识替代对大型视觉语言模型的依赖，实现轻量化端侧GUI智能体：它自主探索应用，构建以UI状态为节点、转移为边的知识图。运行时小模型定位当前节点，在自环、邻接转移、任务完成或回退动作中选择，从而降低推理成本并保护端侧隐私。这种图引导设计减轻了轻量模型的规划负担，同时提升任务效果与可解释性。

WorldMemArena: Evaluating Multimodal Agent Memory Through Action-World Interaction ◆ 相关
· 2026-05-29
WorldMemArena is a benchmark of 400 multi-session multimodal tasks that evaluates agent memory as an Action-World Interaction Loop with a four-stage lifecycle, spanning Lifelong Evolution and Agentic Execution. Fine-grained annotations (gold memory points, updates, distractors, evidence chains) enable stage-level diagnosis and head-to-head comparison of long-context, RAG/external-memory, and harness-based memory agents. Findings show better memory writing does not guarantee better performance, and harness-based memory is flexible but expensive and unreliable.
中文摘要
WorldMemArena是一个包含400个多会话多模态任务的基准，将智能体记忆形式化为具有四阶段生命周期的"动作-世界交互回路"，涵盖终身演化与智能体执行两类任务。其细粒度标注(黄金记忆点、更新、干扰项、证据链)支持阶段级诊断，并实现长上下文、RAG/外部记忆与框架式记忆智能体的同台对比。研究发现更好的记忆写入并不保证更好的性能，框架式记忆虽灵活但成本高且不稳定。

CoHyDE: Iterative Co-Training of LLM Rewriter & Dense Encoder for Tool Retrieval ◆ 相关
· 2026-05-29
CoHyDE tackles tool retrieval over large API catalogs by co-training a dense encoder and an LLM query rewriter as an interdependent system, bridging colloquial queries and technical API vocabulary. The encoder is retrained with InfoNCE on catalog-aware hypothetical descriptions while the rewriter is aligned via DPO using retrieval scores as preference signals. On a ~10k-tool ToolBench subset, three co-training rounds yield +2.5 NDCG@5 on standard queries and up to +8 points on the hardest vague queries.
中文摘要
CoHyDE针对大规模API目录上的工具检索问题，将稠密编码器与LLM查询改写器作为相互依赖的系统进行协同训练，弥合口语化查询与专业API术语之间的鸿沟。编码器用面向目录的假设性描述以InfoNCE损失重训练，改写器则以检索分数为偏好信号通过DPO对齐。在约1万工具的ToolBench子集上，三轮协同训练在常规查询上提升+2.5 NDCG@5，在最难的模糊查询上最高提升约8个百分点。

PhoneWorld: Scaling Phone-Use Agent Environments ◆ 相关
· 2026-05-29
PhoneWorld is a pipeline that converts real GUI trajectories and screenshots into runnable mock Android environments with automatic verification, spanning 34 apps across 16 domains and removing the need to hand-build benchmarks. It automatically recovers which screens matter, how they connect, and which interactions mutate state. Replacing 10K AndroidWorld steps with PhoneWorld supervision improved four benchmarks simultaneously (e.g., +14.7 on AndroidWorld), and broadening app coverage helped more than adding more supervision.
中文摘要
PhoneWorld是一条将真实GUI轨迹与截图转化为可运行的模拟安卓环境(含自动验证)的流水线，覆盖16个领域的34个应用，免去逐一手工构建基准的负担。它能自动识别哪些界面重要、界面如何连接以及哪些交互会改变状态。用PhoneWorld监督替换1万条AndroidWorld步骤后，四个基准同时提升(如AndroidWorld +14.7分)，且在固定预算下扩大应用覆盖比单纯增加监督收益更大。
