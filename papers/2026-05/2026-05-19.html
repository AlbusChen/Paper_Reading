Code as Agent Harness ★ 高度相关
Xuying Ning, Katherine Tieu, Dongqi Fu, Tianxin Wei et al. · 2026-05-19
HF精选multi-agentagent coordination
A survey that reframes code as the operational substrate ("harness") for agent infrastructure, organized into three layers: the harness interface connecting agents to reasoning/action/environment, harness mechanisms (planning, memory, tool use, feedback-driven control), and scaling from single-agent to multi-agent settings. It argues shared code artifacts are the key enabler of multi-agent coordination, review, and verification across domains from coding assistants to enterprise workflows.
中文摘要
本文是一篇综述，将代码重新定位为智能体基础设施的"操作底座"（harness），并按三个层次组织：连接推理/行动/环境的接口层、规划与记忆与工具使用等机制层，以及从单智能体扩展到多智能体的扩展层。作者论证共享代码产物是支撑多智能体协调、审查与验证的关键，覆盖编码助手到企业工作流等多个应用场景。

Agent Bazaar: Enabling Economic Alignment in Multi-Agent Marketplaces ★ 高度相关
Seth Karten, Cameron Crow, Chi Jin · 2026-05-19
HF精选multi-agent
Agent Bazaar is a multi-agent simulation framework for evaluating "Economic Alignment" -- whether collectives of LLM economic agents preserve market stability and integrity. It identifies two emergent failure modes (price-volatility collapse in B2C markets and Sybil deception via coordinated fake identities in C2C markets), shows frontier and open models largely fail to self-regulate, and trains a 9B model with REINFORCE++ and an adaptive curriculum that outperforms all evaluated frontier/open models on the proposed Economic Alignment Score.
中文摘要
Agent Bazaar 是一个多智能体仿真框架，用于评估"经济对齐"——即由大模型经济智能体组成的群体能否维持市场稳定与诚信。论文识别出两种群体涌现失效模式（B2C 市场的价格波动崩盘，以及 C2C 市场中通过协调的虚假身份进行的女巫欺骗），发现前沿与开源模型大多无法自我约束，并用 REINFORCE++ 与自适应课程训练出一个 9B 模型，在所提出的经济对齐分数上超过所有受测前沿/开源模型。

From Runnable to Shippable: Multi-Agent Test-Driven Development for Generating Full-Stack Web Applications from Requirements ★ 高度相关
Yuxuan Wan, Tingshuo Liang, Jiakai Xu, Jingyu Xiao et al. · 2026-05-19
HF精选multi-agent
TDDev is a multi-agent test-driven-development framework that closes the loop for generating full-stack web apps: it converts requirements into acceptance tests before coding, deploys and validates the app via simulated browser interaction, and turns observed failures into structured repair reports for the coding agent. In the first controlled study of TDD protocols for web generation, the TDD infrastructure improves generation quality by 34-48 percentage points over a no-TDD baseline, with the optimal protocol depending on the model's generation style.
中文摘要

Dehai Min, Giovanni Vaccarino, Huiyi Chen, Yongliang Wu et al. · 2026-05-19
HF精选
PUMA is a plug-and-play early-exit framework for large reasoning models that targets overthinking by detecting reasoning-level semantic redundancy -- when successive steps merely revisit established conclusions -- rather than relying only on answer-level confidence signals that may exit prematurely. A lightweight redundancy detector flags candidate exits and answer-level verification confirms safety, removing redundant continuation while preserving answer accuracy and a coherent reasoning prefix across five reasoning models and benchmarks.
中文摘要

This work introduces a model-adaptive definition of tool necessity for LLM agents, grounded in each model's empirical capability rather than treated as a model-agnostic property. SNLP: Layer-Parallel Inference via Structured Newton Corrections ◆ 相关
Ligong Han, Kai Xu, Hao Wang, Akash Srivastava · 2026-05-19
HF精选
SNLP (Structured Newton Layer Parallelism) relaxes the strictly sequential layer dependency of autoregressive Transformers by treating the cross-layer hidden-state trace as a nonlinear residual equation solved with parallel Newton-style updates, replacing expensive exact Jacobians with cheap architecture-induced surrogate dynamics. SNLP-aware regularization improves layer-parallel compatibility and even reduces baseline perplexity by 4.7-23.4% on nanochat-scale Transformers, with practical wall-clock gains at inference.
中文摘要
SNLP（结构化牛顿层并行）通过将跨层隐状态轨迹视为可用并行牛顿式更新求解的非线性残差方程，并以廉价的、由架构诱导的代理动力学替代昂贵的精确雅可比，从而放松自回归 Transformer 严格的逐层依赖。SNLP 感知正则化提升层并行兼容性，甚至在 nanochat 规模 Transformer 上将基线困惑度降低 4.7%–23.4%，并在推理时带来实际墙钟时间收益。
