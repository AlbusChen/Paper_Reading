Solvita: Enhancing Large Language Models for Competitive Programming via Agentic Evolution ★ 高度相关
· 2026-05-18
multi-agentagent coordinationagent framework
Solvita is an agentic evolution framework for hard competitive programming that overcomes the stateless nature of prior multi-agent pipelines by pairing four specialized agents (Planner, Solver, Oracle, Hacker) with trainable graph-structured knowledge networks. Outcome signals (pass/fail verdicts, test certification quality, discovered vulnerabilities) are recast as RL updates that let agents dynamically route future queries from accumulated experience, without updating LLM weights. It sets a new SOTA across CodeContests, APPS, AetherCode, and live Codeforces rounds, nearly doubling single-pass accuracy.
中文摘要
Solvita 提出面向高难度竞赛编程的智能体进化框架，针对现有多智能体流水线“无状态”、丢弃历史经验的问题，为 Planner、Solver、Oracle、Hacker 四个专用智能体各配备可训练的图结构知识网络。系统将通过/失败判定、测试认证质量、Hacker 发现的对抗漏洞等结果信号转化为强化学习更新，使智能体能基于历史成败动态路由后续查询，且无需更新底层 LLM 权重。在 CodeContests、APPS、AetherCode 及 Codeforces 实时比赛上刷新代码生成智能体 SOTA，单次通过准确率几乎翻倍。

Agentic Discovery of Neural Architectures: AIRA-Compose and AIRA-Design ★ 高度相关
· 2026-05-18
multi-agentagent frameworkefficiency
From Meta FAIR, this work shows LLM agents can autonomously discover foundation-model architectures beyond Transformers via two multi-agent frameworks: AIRA-Compose (11 agents searching computational primitives under a 24-hour budget) and AIRA-Design (20 agents writing novel attention mechanisms and training scripts). The discovered AIRAformer/AIRAhybrid families, pretrained at 1B scale, outperform Llama 3.2 by 2.4-3.8% on downstream tasks and scale 54-71% faster, presenting a concrete step toward recursive self-improvement.
中文摘要
本文（来自 Meta FAIR）展示了 LLM 智能体可自主发现超越标准 Transformer 的基础模型架构，提出两个多智能体框架：AIRA-Compose（11 个智能体在 24 小时预算内搜索计算原语）与 AIRA-Design（20 个智能体编写新型注意力机制与训练脚本）。所发现的 AIRAformer/AIRAhybrid 系列在 1B 规模预训练后，下游任务准确率较 Llama 3.2 提升 2.4-3.8%，扩展速度快 54-71%，是迈向递归自我改进的实质性一步。

PhysBrain 1.0 Technical Report · 低相关
· 2026-05-18
Tech Report
PhysBrain 1.0 is a technical report on building physical-commonsense priors for vision-language-action (VLA) robot models by converting large-scale human egocentric video into structured QA supervision (scene elements, spatial dynamics, action execution, depth-aware relations), then transferring these priors to VLA policies via a capability-preserving, language-sensitive adaptation. It achieves SOTA across multimodal QA and embodied control benchmarks (ERQA, PhysBench, SimplerEnv-WidowX, LIBERO, RoboCasa) with strong out-of-domain results. Relevance to multi-agent efficiency/communication is weak; included as a technical report.
中文摘要
PhysBrain 1.0 技术报告研究为视觉-语言-动作（VLA）机器人模型构建物理常识先验：将大规模人类第一视角视频转化为结构化问答监督（场景要素、空间动态、动作执行、深度关系），再通过保能力、对语言敏感的适配方法迁移到 VLA 策略。在 ERQA、PhysBench、SimplerEnv-WidowX、LIBERO、RoboCasa 等多模态问答与具身控制基准上取得 SOTA，并展现较强域外泛化。注：与多智能体效率/通信主题相关性较弱，作为技术报告收录。

Look Before You Leap: Autonomous Exploration for LLM Agents ◆ 相关
· 2026-05-18
llm agentexploration
This paper identifies autonomous exploration as a critical missing capability for LLM agents, which tend to prematurely exploit prior knowledge in unfamiliar environments. It introduces Exploration Checkpoint Coverage as a verifiable metric and a training strategy that interleaves task-execution and exploration rollouts with separate verifiable rewards, yielding the 'Explore-then-Act' paradigm that decouples information gathering from task execution. Results show systematic exploration improves generalization for real-world-ready agents.
中文摘要
本文指出自主探索是 LLM 智能体被忽视的关键能力——智能体在陌生环境中往往过早利用先验知识。作者提出可验证指标 Exploration Checkpoint Coverage，并设计交替进行任务执行与探索 rollout、分别用可验证奖励优化的训练策略，由此提出“先探索后行动”（Explore-then-Act）范式，将信息收集与任务执行解耦。实验表明系统化探索能显著提升智能体的泛化与真实环境适应能力。

MMSkills: Towards Multimodal Skills for General Visual Agents ◆ 相关
· 2026-05-18
agent frameworkllm agent
MMSkills introduces a framework for representing, generating, and using reusable multimodal procedural knowledge for visual agents, where each skill couples a textual procedure with runtime state cards and multi-view keyframes. An agentic trajectory-to-skill generator converts public interaction traces into skills, and a branch-loaded agent consults visual evidence in a temporary branch to avoid excessive image context. Experiments on GUI and game benchmarks show consistent gains for both frontier and smaller multimodal agents.
中文摘要
MMSkills 提出面向视觉智能体的多模态过程性知识表示、生成与使用框架：每个技能将文本流程与运行时状态卡片、多视角关键帧耦合。其智能体式“轨迹到技能”生成器将公开交互轨迹转化为可复用技能，并通过分支加载式智能体在临时分支中查阅视觉证据，避免过多图像上下文开销。在 GUI 与游戏类基准上，对前沿及小型多模态智能体均取得稳定提升。

Distilling Long-CoT Reasoning through Collaborative Step-wise Multi-Teacher Decoding ◆ 相关
· 2026-05-18
collaborative decodingmulti-teacherefficiency
CoRD is a collaborative multi-teacher decoding framework for distilling Long-CoT reasoning, addressing the redundant sampling and missed complementarity of post-hoc trace curation. It performs step-wise reasoning synthesis guided by predictive perplexity scoring and beam search, letting heterogeneous large reasoning models jointly construct coherent trajectories. CoRD yields higher-quality reasoning data and near teacher-level student performance with fewer supervision signals and little efficiency overhead, generalizing to out-of-domain settings.
中文摘要

PAGER: Bridging the Semantic-Execution Gap in Point-Precise Geometric GUI Control ◆ 相关
· 2026-05-18
gui agent
PAGER targets precision-sensitive GUI tasks where actions must land on exact points in continuous canvas space and local coordinate errors cascade into topological failures. It introduces PAGE Bench (4,906 problems, 224K pixel-level actions) and a topology-aware agent that decomposes construction into dependency-structured planning and pixel-level execution, combining pixel-grounded supervised tuning with precision-aligned RL. PAGER delivers 4.1x higher task success than the strongest general baseline and raises step success from below 9% to over 62% for GUI-specialized agents.
中文摘要
PAGER 针对“精度敏感”的 GUI 任务——动作须精确落在连续画布空间中的点，局部坐标误差会级联引发拓扑失败。论文提出 PAGE Bench（4,906 个问题、22.4 万像素级动作）以及一个拓扑感知智能体，将几何构造分解为依赖结构化规划与像素级执行，结合像素对齐的监督微调与精度对齐的强化学习。PAGER 的任务成功率较最强通用基线提升 4.1 倍，并将 GUI 专用智能体的单步成功率从不足 9% 提升至 62% 以上。

GQLA: Group-Query Latent Attention for Hardware-Adaptive Large Language Model Decoding ◆ 相关
· 2026-05-18
llm efficiencykv cacheinference
GQLA (Group-Query Latent Attention) is a minimal modification of DeepSeek's Multi-head Latent Attention whose trained weights expose two algebraically equivalent decoding paths over the same parameters: an MQA-absorb path (H100-optimal) and a GQA path with per-group expanded cache (commodity-GPU and MTP-friendly), selectable at runtime without retraining or custom kernels. A TransGQLA conversion turns pretrained GQA checkpoints into GQLA models, compressing LLaMA-3-8B per-token KV cache to 28.125% of the GQA baseline while supporting up to 8-way tensor parallelism.
中文摘要
GQLA（分组查询潜在注意力）是对 DeepSeek 多头潜在注意力（MLA）的最小化改造：同一套训练权重暴露两条代数等价的解码路径——MQA-absorb 路径（适配 H100）与按组扩展缓存的 GQA 路径（适配商用 GPU 与多 token 预测），运行时按硬件选择，无需重训练或定制算子。配套 TransGQLA 可将预训练 GQA 检查点转换为 GQLA 模型，在 LLaMA-3-8B 上将每 token KV 缓存压缩至 GQA 基线的 28.125%，并支持最高 8 路张量并行。

Known By Their Actions: Fingerprinting LLM Browser Agents via UI Traces ◆ 相关
· 2026-05-18
llm agentbrowser agentsecurity
This security study shows that websites can passively fingerprint which LLM powers a browsing agent: across 14 frontier LLMs and four web environments, a passive JavaScript tracker capturing agent actions and interaction timings identifies the underlying model with up to 96% F1. Classifiers generalize across model sizes and families, need few traces, and infer identity early in an episode; randomized timing delays degrade but do not robustly prevent the attack, since classifiers retrained on delayed traces largely recover.
中文摘要
这项安全研究表明，网站可被动指纹识别驱动浏览智能体的底层 LLM：在 14 个前沿 LLM 与四类 Web 环境中，仅靠被动 JavaScript 追踪器记录的智能体动作与交互时序，即可以高达 96% F1 识别底层模型。分类器可跨模型规模与系列泛化，仅需少量轨迹，并能在交互早期推断身份；注入随机时序延迟虽降低识别率，但无法稳健防御——在延迟轨迹上重训练的分类器可大体恢复性能。
