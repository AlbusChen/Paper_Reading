Maestro: Reinforcement Learning to Orchestrate Hierarchical Model-Skill Ensembles ★ 高度相关
· 2026-05-22
Existing agent frameworks use a monolithic LLM and fixed logic, failing to exploit the complementary strengths of different models and skills. Maestro is an RL-driven orchestration framework that reframes multimodal tasks as sequential decision-making over a hierarchical model-skill registry, training a lightweight policy to compose ensembles of frozen experts via outcome-based rewards. A 4B orchestrator reaches 70.1% average accuracy across ten multimodal benchmarks, surpassing GPT-5 and Gemini-2.5-Pro, and the learned coordination policy generalizes to unseen models and skills without retraining.
中文摘要
现有智能体框架依赖单一LLM与固定逻辑，无法发挥不同模型与技能的互补优势。本文提出Maestro，一种由强化学习驱动的编排框架，将多模态任务建模为对分层“模型-技能”注册表的序贯决策，通过基于结果的奖励训练轻量策略动态组合冻结的专家模型。仅4B的编排器在十个多模态基准上取得70.1%平均准确率，超越GPT-5与Gemini-2.5-Pro，且所学协调策略无需重训即可泛化到未见过的模型与技能。

Efficient Agentic Reasoning Through Self-Regulated Simulative Planning ★ 高度相关
· 2026-05-22
Reactive agentic LLMs with implicit chain-of-thought planning waste tokens without reliable accuracy gains. SR2AM decomposes decision-making into simulative reasoning (a learned world model), self-regulation (a configurator deciding when and how deeply to plan), and reactive execution, trained via SFT then RL. An 8B/30B model matches 120B-1T-scale systems on math, science, tabular, and web tasks while using 25.8-95.3% fewer reasoning tokens; RL learns to plan farther ahead (+22.8% horizon) rather than more often (+2.0% frequency).
中文摘要
本文针对反应式智能体LLM隐式规划导致推理token冗长却无稳定精度收益的问题，提出SR2AM框架，将决策分解为基于世界模型的模拟推理、决定何时及多深规划的自调节模块、以及反应式执行三层，并通过监督学习与强化学习训练。8B/30B模型在数学、科学、表格分析与网页检索任务上达到120B-1T级系统的水平，同时减少25.8%-95.3%的推理token；强化学习使其学会规划得更远（规划跨度+22.8%）而非更频繁（频率仅+2.0%）。

One Sentence, One Drama: Personalized Short-Form Drama Generation via Multi-Agent Systems ◆ 相关
· 2026-05-22
multi-agent
Short-drama generation pipelines built on one-shot LLM scripts struggle with narrative pacing, spatial consistency, and quality control. This paper presents a hierarchical multi-agent framework that turns a single-sentence idea into a full short drama via a multi-agent debate story module, a 3D-grounded first-frame mechanism for cross-clip spatial consistency, and multi-stage reviewer loops for error detection. On the new Short-Drama-Bench it significantly outperforms existing pipelines in narrative quality and cross-clip consistency.
中文摘要
现有短剧生成流程基于一次性LLM脚本，难以满足叙事节奏、空间一致性与质量控制三项要求。本文提出层级式多智能体框架，通过多智能体辩论的故事生成模块、保证跨片段空间一致的3D对齐首帧机制、以及多阶段审查循环，将一句话创意转化为完整短剧。在新提出的Short-Drama-Bench上，该方法在叙事质量与跨片段一致性方面显著优于现有流程。

Rule2DRC: Benchmarking LLM Agents for DRC Script Synthesis with Execution-Guided Test Generation ◆ 相关
· 2026-05-22
Translating natural-language chip design rules into executable DRC scripts is labor-intensive, motivating LLM agents, but existing benchmarks are small and score by code similarity rather than execution. Rule2DRC is a large-scale benchmark with 1,000 rule-to-script tasks and 13,921 chip layouts for execution-based scoring without needing test layouts as agent input. The paper also proposes SplitTester, a tester agent that uses execution feedback to generate discriminative test cases, substantially improving Best-of-N script selection.
中文摘要
将自然语言芯片设计规则转译为可执行的DRC脚本耗时且需专业知识，促使研究者引入LLM智能体，但现有基准规模小且以代码相似度而非执行正确性评分。本文提出Rule2DRC，一个大规模基准，含1,000个规则到脚本任务与13,921个芯片版图，支持无需将测试版图作为智能体输入的执行式评分。论文还提出SplitTester——一个利用执行反馈生成区分性测试用例的测试智能体，显著提升Best-of-N脚本选择性能。

π-Bench: Evaluating Proactive Personal Assistant Agents in Long-Horizon Workflows ◆ 相关
· 2026-05-22
Personal assistant agents must provide proactive help, since users start from underspecified requests with unstated needs, yet existing benchmarks rarely test acting on hidden intents across sustained interactions. Pi-Bench introduces 100 multi-turn tasks across 5 user personas, incorporating hidden intents, inter-task dependencies, and cross-session continuity to jointly measure proactivity and task completion. Experiments show proactive assistance remains hard, that it is distinct from task completion, and that prior interactions help resolve intent in later tasks.
中文摘要
个人助理智能体需提供主动协助，因为用户往往以欠明确的请求开始、并留有未言明的需求，但现有基准很少评测在持续交互中识别并响应隐含意图的能力。本文提出π-Bench，包含5种用户人设下的100个多轮任务，融入隐含意图、任务间依赖与跨会话连续性，联合衡量主动性与任务完成度。实验表明主动协助仍具挑战，主动性与任务完成度明显不同，且先前交互有助于后续任务中的意图推断。

ACC: Compiling Agent Trajectories for Long-Context Training ◆ 相关
· 2026-05-22
Agents generate massive multi-turn trajectories, but standard agent SFT masks tool responses and only trains turn-level tool selection, leaving scattered cross-turn evidence unused. Agent Context Compilation (ACC) converts trajectories from search, software-engineering, and database agents into long-context QA pairs that pair the original question with tool responses gathered across turns, giving direct supervision for long-context reasoning without extra annotation. Training Qwen3-30B with ACC reaches 68.3 on MRCR (+18.1) and 77.5 on GraphWalks (+7.6), comparable to Qwen3-235B while preserving general capabilities.
中文摘要
智能体在解题时产生大量多轮轨迹，但标准智能体SFT会掩盖工具响应、仅训练轮级工具选择，使分散于各轮的证据信号未被利用。本文提出智能体上下文编译（ACC），将搜索、软件工程与数据库智能体的轨迹转化为长上下文问答对，把原始问题与跨轮收集的工具响应组合起来，无需额外标注即可直接监督长上下文推理。以ACC训练的Qwen3-30B在MRCR上达68.3（+18.1）、GraphWalks上达77.5（+7.6），媲美Qwen3-235B且保持通用能力。

Spreadsheet-RL: Advancing Large Language Model Agents on Realistic Spreadsheet Tasks via Reinforcement Learning ◆ 相关
· 2026-05-22
Most spreadsheet agents rely on prompting general-purpose LLMs and struggle with complex multi-step workflows. Spreadsheet-RL is an RL fine-tuning framework that trains specialized agents in a realistic Excel environment, with an automated pipeline for collecting paired start-goal spreadsheets, a new Domain-Spreadsheet benchmark, and a multi-turn 'Spreadsheet Gym' exposing Excel via a Python sandbox. GenEvolve: Self-Evolving Image Generation Agents via Tool-Orchestrated Visual Experience Distillation ◆ 相关
· 2026-05-22
High-quality open-ended image generation increasingly needs an agent to combine a model's generative ability with external tools. GenEvolve is a self-evolving framework based on Tool-Orchestrated Visual Experience Distillation: each generation is a tool-orchestrated trajectory, and instead of scalar image rewards it compares multiple trajectories for the same request, abstracting best-worst differences into structured visual experience that gives a teacher branch dense token-level supervision. On public benchmarks and the new GenEvolve-Bench it achieves state-of-the-art results among image-generation frameworks.
中文摘要
高质量的开放式图像生成日益需要智能体将模型自身的生成能力与外部资源结合。本文提出GenEvolve，一个基于“工具编排的视觉经验蒸馏”的自进化框架：每次生成被建模为工具编排轨迹，区别于依赖标量图像奖励的方法，它对同一请求比较多条轨迹，将最优-最差差异抽象为结构化视觉经验，为特权教师分支提供稠密的token级监督。在公开基准与新提出的GenEvolve-Bench上，该方法在图像生成框架中取得最先进性能。

KVServe: Service-Aware KV Cache Compression for Communication-Efficient Disaggregated LLM Serving ◆ 相关
· 2026-05-22
Disaggregated LLM serving turns the KV cache into an explicit payload crossing network and storage boundaries, making KV communication a dominant end-to-end bottleneck that static compression configs handle poorly. KVServe is a service-aware, adaptive KV communication-compression framework: it unifies compression methods into a recomposable strategy space, uses a Bayesian Profiling Engine to distill a Pareto candidate set (50x less offline search), and a Service-Aware Online Controller to pick profiles under live constraints. Integrated into vLLM, it achieves up to 9.13x job-completion-time speedup and 32.8x TTFT reduction.
中文摘要
分离式LLM服务将KV缓存变为跨网络与存储边界传输的显式负载，使KV通信成为端到端主要瓶颈，而静态压缩配置难以应对。本文提出KVServe——首个面向服务、自适应的KV通信压缩框架：将压缩方法统一为可重组策略空间，用贝叶斯剖析引擎蒸馏帕累托候选集（离线搜索开销降低50倍），并以服务感知在线控制器在实时约束下选择配置。集成到vLLM后，最高实现9.13倍作业完成时间加速与32.8倍首token时延降低。

TerminalWorld: Benchmarking Agents on Real-World Terminal Tasks ◆ 相关
· 2026-05-22
Existing terminal-agent benchmarks are small and often score scripts by code similarity rather than execution. TerminalWorld is a scalable data engine that reverse-engineers evaluation tasks from 80,870 in-the-wild terminal recordings, yielding 1,530 validated tasks across 18 categories plus a 200-task manually verified subset. Benchmarking eight frontier models and six agents shows a maximum 62.5% pass rate and only weak correlation (r=0.20) with expert-curated benchmarks like Terminal-Bench.
中文摘要
现有终端智能体基准规模小，且常以代码相似度而非执行结果评分。本文提出TerminalWorld，一个可扩展的数据引擎，从80,870条真实终端录制中逆向构建评测任务，得到覆盖18个类别的1,530个验证任务及200个人工核验子集。对八个前沿模型与六个智能体的评测显示最高通过率仅62.5%，且与Terminal-Bench等专家构建基准仅弱相关（r=0.20），表明其捕捉了不同维度的真实能力。

Lean Refactor: Multi-Objective Controllable Proof Optimization via Agentic Strategy Search ◆ 相关
· 2026-05-22
LLM-generated Lean proofs are correct but verbose and brittle across library versions. Lean Refactor is a plug-and-play retrieval-augmented agentic framework that steers a frozen LLM with retrievals from a curated database of multi-objective refactoring strategies annotated with supported Lean/Mathlib versions and expected compilation-cost reduction, avoiding retraining on each model or Lean release. It achieves over 70% token compression on competition benchmarks, 20%+ on research repos, and up to 60% compilation-time reduction, outperforming prior work and Claude Code.
中文摘要
LLM生成的Lean证明虽正确却冗长，且在不同库版本间易失效。本文提出Lean Refactor，一个即插即用的检索增强智能体框架：以检索结果引导冻结的LLM，检索库来自经精心标注的多目标重构策略数据库（标注支持的Lean/Mathlib版本与预期编译开销降幅），从而避免随每次模型或Lean版本更新而重训。该方法在竞赛基准上实现70%以上的token压缩、在科研仓库上达20%以上，并实现最高60%的编译时间降低，优于既有工作与Claude Code。
